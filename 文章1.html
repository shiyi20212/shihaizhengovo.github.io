HTML实现个人博客网页的完整教程

发布时间：2026-01-07 分类：前端开发 阅读量：128

本文详细讲解了如何用HTML+CSS制作一份完整的个人博客网页，包含导航栏、Banner、博客列表、侧边栏、页脚等核心模块，兼顾响应式适配，新手也能轻松上手...

一、HTML与CSS：网页的“骨架”与“外衣”

HTML（超文本标记语言）是网页的“骨架”，负责定义网页的结构（比如标题、段落、按钮）；CSS（层叠样式表）是网页的“外衣”，负责控制网页的外观（比如颜色、布局、大小）。 要制作博客网页，需要用HTML搭建结构，再用CSS美化样式。

二、核心模块1：导航栏（网页的“指引牌”）

导航栏是博客的“指引牌”，通常包含“首页、博客、关于我”等选项，需要固定在页面顶部方便用户点击。 用HTML定义导航结构，CSS实现固定定位和样式：

<!-- 导航栏HTML结构 --> <nav class="nav-bar"> <div class="logo">我的博客</div> <ul class="nav-links"> <li><a href="#">首页</a></li> <li><a href="#">博客</a></li> <li><a href="#">关于我</a></li> </ul> </nav> <!-- 导航栏CSS样式 --> .nav-bar { background: #fff; box-shadow: 0 2px 8px rgba(0,0,0,0.1); position: sticky; /* 固定在顶部 */ top: 0; padding: 15px 20px; display: flex; justify-content: space-between; align-items: center; }

三、核心模块2：Banner（博客的“门面”）

Banner是博客的“门面”，通常展示博客标题和简介，用背景图+文字的组合提升视觉效果：

<!-- Banner HTML结构 --> <section class="banner"> <div class="banner-text"> <h1>我的个人博客</h1> <p>记录成长，分享所学</p> </div> </section> <!-- Banner CSS样式 --> .banner { height: 400px; background: url("banner.jpg") center/cover; /* 背景图 */ display: flex; align-items: center; justify-content: center; color: #fff; text-align: center; }

四、让博客适配手机：响应式布局

为了让博客在手机上也能正常显示，需要用CSS的“媒体查询”功能，在小屏幕设备上自动调整布局（比如把导航栏改成“汉堡菜单”）：

<!-- 响应式CSS --> @media (max-width: 768px) { /* 手机端隐藏导航列表，显示汉堡按钮 */ .nav-links { display: none; } .hamburger { display: block; } /* 缩小Banner高度 */ .banner { height: 280px; } }

通过以上步骤，就能完成一个包含“导航栏、Banner、博客列表”的完整博客网页啦！HTML负责搭结构，CSS负责美化和适配，新手也能快速上手~
