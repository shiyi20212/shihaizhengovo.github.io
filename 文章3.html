移动端汉堡菜单的实现思路

发布时间：2026-01-05 分类：前端交互 阅读量：156

汉堡菜单是移动端导航的常用组件，本文结合HTML+CSS+JS实现了点击切换的汉堡菜单，包含滑入动画、大屏适配等细节，提升移动端用户体验...

一、汉堡菜单：移动端导航的“压缩包”

汉堡菜单是由“三条横线”组成的导航按钮，点击后会展开隐藏的导航列表——它能节省移动端的屏幕空间，是适配小屏设备的核心交互组件之一。

二、步骤1：搭建HTML结构（按钮+菜单）

汉堡菜单的核心结构是“触发按钮（三条横线）”+“导航列表（隐藏）”，用HTML定义这两个部分：

<!-- 汉堡按钮：触发菜单显示 --> <div class="hamburger" id="hamburgerBtn"> <span></span> <span></span> <span></span> </div> <!-- 导航菜单：默认隐藏 --> <nav class="nav-menu" id="navMenu"> <ul> <li><a href="#">首页</a></li> <li><a href="#">博客</a></li> <li><a href="#">关于我</a></li> </ul> </nav>

三、步骤2：CSS控制样式与动画

用CSS实现“按钮样式”“菜单隐藏”“点击动画”三个核心效果：

/* 汉堡按钮样式 */ .hamburger { width: 30px; height: 24px; display: flex; flex-direction: column; justify-content: space-between; cursor: pointer; } .hamburger span { display: block; width: 100%; height: 4px; background: #333; border-radius: 2px; transition: all 0.3s; /* 动画过渡 */ } /* 菜单默认隐藏（右侧滑出） */ .nav-menu { position: fixed; top: 0; left: 0; width: 200px; height: 100vh; background: #fff; box-shadow: 0 2px 8px rgba(0,0,0,0.2); transform: translateX(-100%); /* 移出可视区 */ transition: transform 0.3s; /* 滑入动画 */ } /* 激活状态：按钮变叉号+菜单滑入 */ .hamburger.active span:nth-child(1) { transform: translateY(10px) rotate(45deg); } .hamburger.active span:nth-child(2) { opacity: 0; } .hamburger.active span:nth-child(3) { transform: translateY(-10px) rotate(-45deg); } .nav-menu.active { transform: translateX(0); /* 滑入可视区 */ }

效果演示：点击下方“汉堡按钮”

  • 首页
  • 博客
  • 关于我

四、步骤3：JS实现点击切换

用JavaScript给按钮绑定点击事件，切换“激活状态类”，实现菜单的显示/隐藏：

// 获取元素 const hamburgerBtn = document.getElementById('hamburgerBtn'); const navMenu = document.getElementById('navMenu'); // 点击按钮切换激活类 hamburgerBtn.addEventListener('click', () => { hamburgerBtn.classList.toggle('active'); navMenu.classList.toggle('active'); }); // 可选：点击菜单项后关闭菜单 const menuItems = document.querySelectorAll('.nav-menu a'); menuItems.forEach(item => { item.addEventListener('click', () => { hamburgerBtn.classList.remove('active'); navMenu.classList.remove('active'); }); });

五、大屏适配：自动切换导航样式

用CSS媒体查询，在大屏设备上自动隐藏汉堡按钮、显示普通导航：

@media (min-width: 768px) { /* 大屏隐藏汉堡按钮 */ .hamburger { display: none; } /* 大屏显示横向导航 */ .nav-menu { position: static; transform: translateX(0); width: auto; height: auto; box-shadow: none; } .nav-menu ul { display: flex; gap: 20px; } }
